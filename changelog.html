Changelog

A record of all notable changes made to LuxPDF. We are constantly working to improve our tools and add new features.

Version 1.2 - July 2025

  • New: Added a public changelog to track all future updates!
  • Fixed: Completely rebuilt the PDF to PNG/JPEG converters. They now correctly render high-quality images.
  • Fixed: Implemented real PDF compression. The tool now properly reduces file size where possible.
  • Fixed: The PDF to TXT tool is now fully functional.
  • Infrastructure: Migrated the entire website from Netlify to Cloudflare Pages for improved performance, scalability, and unlimited free bandwidth.
  • Infrastructure: Added all necessary PDF processing libraries (`pdf-lib`, `pdf.js`) directly to the site for improved reliability.

Version 1.1 - July 2025

  • New: Added a custom favicon to match the brand's logo and color scheme.
  • SEO: Implemented SEO-optimized meta descriptions and title tags across the site.
  • SEO: Created and submitted a `sitemap.xml` to search engines.
  • Monetization: Integrated Google AdSense script.

Version 1.0 - Initial Release

  • Initial launch of LuxPDF with core tools: Merge, Split, Rotate, Compress, and Convert.
